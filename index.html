    Projects

    Browse my projects

    WanderWise — iOS Travel Planner

    Swift app with three tabs: Discover (API powered destinations), Favorites (saved places), and My Trips (plans). Includes AI travel advice for each destination. Focus on networking, data passing, persistence, and a simple clean design.

    Personal Blog — Next.js, Sanity, React, TypeScript

    Full stack blog with Sanity CMS. Supports rich posts, images, and planned comment features and image carousels. Built to master modern web tools and content workflows.

    Pipelined MIPS CPU — FPGA

    Five stage pipeline with forwarding and stalling. Includes support for atomic instructions. Built test benches to validate functional correctness and timing on FPGA boards.

    Sobel Edge Detection — Hardware Accelerator

    Custom accelerator for Sobel filtering with line buffered pipelines. Achieved real time edge detection on video input. Validated with synthesis and on board tests.

    ARM NEON Video Filters

    Single and multithreaded grayscale and Sobel filters optimized with ARM NEON intrinsics for real time performance. Benchmarked throughput and latency across cores.

    Programmable Bike Light — Verilog

    Beat32 timing base, timer, and shifter modules with a master FSM and programmable blinker. Uses defined D flip flops for clean sequential logic. Reset starts at one second per design choice.

    Mango Pi Curtain Opener — Bare Metal C

    Wrote RTC and I2C drivers to schedule curtain movement. IR remote control library and simple shell for commands. Partner project with focus on reliability and timing.

    Dynamic Music Player — FPGA

    Custom modules for chords, harmonics, echo, dynamics, and note display. Pipelined waveform generation with VGA rendering. Final project for EE108.

    Enigma Machine Simulation — JavaScript

    Implemented classic rotor based encryption and decryption logic. Built a simple UI to configure rotors and visualize letter mappings for learning and demos.

    Egg Incubator — Arduino C++

    Maintains temperature and humidity with sensors and control loops. Buzzer for high temperature, fan control for cooling, and humidifier or evaporation to adjust humidity.
